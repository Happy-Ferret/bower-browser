# bower-browser (WIP)

> GUI tool for managing Bower components.

## Installation
```shell
$ npm install -g bower-browser
```

## Usage
```shell
$ cd path/to/your-project
$ bower-browser
```

Then, browser will open <http://localhost:3000> automatically.  
Manage your Bower components in the web GUI! :-)
